# RPG Microservices Learning

Este repositorio contiene el código y la documentación para el tutorial de Microservicios con Go y Kafka.

## Estructura del Proyecto

La metodología de este curso utiliza "proyectos por sección". Cada carpeta dentro de `projects/` representa el estado final del código para esa sección específica.

### Proyectos Disponibles

- **[Sección 02: Conceptos Básicos](projects/section-02-kafka-basics/)**: Código introductorio con productores y consumidores básicos.

### Documentación

La guía paso a paso se encuentra en la carpeta `tutorial/`.

- [02. Conceptos de Kafka y Go](tutorial/02-conceptos-kafka-go.md)
